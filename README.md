# PC Boost Script 🚀

This script helps improve system performance by cleaning temporary files, flushing DNS cache, and running disk cleanup.

## Features
- Cleans user temp files
- Cleans Windows temp files
- Flushes DNS cache
- Runs disk cleanup

## How to Use
1. Download boost.bat
2. Right click → Run as Administrator

## Important
This script improves performance but does not replace hardware upgrades.
